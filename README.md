# WorkNest ERP

## Tech Stack
- PHP 8.x (Core PHP)
- MySQL
- Apache (XAMPP)
- Bootstrap 5
- HTML5 / CSS3

## Features Implemented
- Project structure setup
- Database schema with relationships
- Basic configuration setup

## Setup Instructions
1. Install XAMPP
2. Start Apache and MySQL
3. Place project inside:
   C:\xampp\htdocs\
4. Create database:
   worknest_db
5. Import:
   database/schema.sql
6. Open in browser:
   http://localhost/worknest-erp
